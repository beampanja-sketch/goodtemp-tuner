# GOODTEMP TUNER

เครื่องมือช่างสำหรับตรวจระบบแอร์รถยนต์และรถห้องเย็น คำนวณค่า Subcool / Superheat จากค่าความดัน (PSI) และอุณหภูมิทางออก เพื่อช่วย:

- เช็คปริมาณน้ำยา (จาก Subcool)
- ปรับวาล์ว TXV (จาก Superheat)
- วินิจฉัยอาการระบบ (น้ำยาขาด/เกิน, อุดตัน, คอมพ์อ่อน ฯลฯ)

**รองรับ:** R-134a (แอร์รถยนต์) และ R-404A (รถห้องเย็น)

## วิธีใช้

ใส่ค่าจากเกจ์และเทอร์โมมิเตอร์ — ใส่ฝั่งใดฝั่งหนึ่งก็ได้ ไม่ต้องครบทั้ง 4 ค่า:

- **ฝั่งต่ำ (LP + คอยล์เย็น)** → เช็ควาล์ว TXV
- **ฝั่งสูง (HP + คอยล์ร้อน)** → เช็คน้ำยา
- **ทั้งสองฝั่ง** → วินิจฉัยเต็มระบบ
